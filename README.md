# AEDS2
Algoritmos desenvolvidos na disciplina de algoritmos e estrutura de dados 2 

## Os tópicos foram abordados nas linguagens C e Java. O conteúdo do material se divide da seguinte maneira:
  1. Introdução, prática de algoritmos interativos e recursivos
  2. Estrutura de dados básicas (Fila,Pilha,Lista) do tipo sequencial
  3. Algoritmos de pesquisa e ordenação
  4. Estrutura de dados do tipo flexível - uso de ponteiros - Fila, Pilha, Lista, Lista duplamente Encadeada, Matriz
  5. Árvore binárias, AVL, Rubronegra, tabela HASH, Hash indireto, Hash parfeito, Hash com área de reserva, Árvore TRIE, Estruturas Híbridas
